# API Reviews with Laravel

Se trata simplemente de un proyecto de práctica el cual tiene la finalidad de replicar un sistema de reviews de productos de un teórico Ecommerce. Los usuarios que sean identificados como 'customers' pueden hacer reseñas, editarlas o borrarlas. Tan solo pueden hacer una reseña por producto y tan solo pueden modificar sus propias reseñas.

Las rutas para introducir, modificar o destruir información están protegidas con Laravel Sanctum, mientras que las consultas de reviews o de productos son públicas.
